# design-patterns

design patterns 理解设计模式

## Typescript Deno Start

Install [Deno][https://deno.land/]

```typescript

    // start
    deno run {fileName}

    class Node {
        // 私有属性以 `#` 开头
        #size: number = 0;
        get size() {
            return this.$size;
        }
        // 私有方法以 `$` 开头
        private $getCode() {}
    }

```
